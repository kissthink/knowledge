ios使用技巧
==================

常见命令::

  <shift> + <cmd> + k     // 清理缓存
  <cmd> + ,   // 打开配置
  Window -> Devices    // 打开设备管理


新加测试机操作::

  1. 打开develop网站,在device中增加一条记录(udid可以通过“设备管理器”查到)
  2. <cmd> + , 打开配置，点击“Accounts”
  3. 找到指定帐号, 双击打开
  4. 点击左下角的刷新与线上进行同步
  5. 完成后<shift> + <cmd> + k进行缓存清理
  6. done
